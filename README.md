# 🎨 Paint — рисовалка в браузере

Простая онлайн-рисовалка в стиле минимализма. Никаких зависимостей, чистый HTML + CSS + JavaScript.

## Возможности

- Кисть и ластик
- Выбор любого цвета (палитра + произвольный)
- Регулировка толщины линии
- Очистка холста
- Сохранение работы в PNG
- Работа с мышью и сенсорным экраном

## Запуск локально

Просто открой `index.html` в браузере — никакой сборки не требуется.

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи в него все файлы (`index.html`, `styles.css`, `script.js`).
3. Перейди в **Settings → Pages**.
4. В разделе **Source** выбери ветку `main` и папку `/ (root)`.
5. Через минуту сайт будет доступен по адресу `https://<твой-логин>.github.io/<имя-репозитория>/`.

## Структура

```
├── index.html   # разметка
├── styles.css   # стили
└── script.js    # логика рисования
```
